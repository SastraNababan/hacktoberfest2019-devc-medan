# Strpos

Fungsi strpos() mencari dan mencocokkan satu atau dua kata dalam satu kalimat kedalam variabel yg berisi string yg di tentukan.

Catatan: Fungsi strpos() hanya bisa melakukan pengecekan jika array hanya berisi satu nilai string, maka untuk melakukan pengecekan kedalam array yg lebih dari satu, diperlukan fungsi tambahan yaitu foreach().

### Deskripsi

```
strpos(string,stringToFind,start)
```

### Parameter

string
&nbsp;&nbsp;Required. adalah variabel yg berisi string tempat kata yg mau dicari
array  
&nbsp;&nbsp;Required. variabel yg berisi string yg akan dicari
start
&nbsp;&nbsp;Optional. posisi karakter untuk mulai pencarian
